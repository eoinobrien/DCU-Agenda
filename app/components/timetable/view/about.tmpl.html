About DCU Timetables

Created by Eoin O'Brien, using an API created for his Third Year Project. Built using an Angular frontend, and a PHP API that connects to DCU timetables, and interprets the returned data into a JSON format. The data is also cached so multiple request can be served quickly.

Made to replace the fantastic application DCUTT which was created by Ian Duffy and Ben Chapman.

Got any ideas or fixes submit them on Github. To contact me, drop me an email.
